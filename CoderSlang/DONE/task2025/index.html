HTML

HTML stands for HyperText Markup Language. HTML is responsible for the structure and content of a page, and tags are the building blocks for building a page.

Tags structure

All tags have names that are enclosed in angle brackets, or, in other words, signs less than `<` and greater than `>`

Paired tags

Paired tags have two parts: an opening tag and a closing tag. The closing tag is distinguished by the presence of a `/` before the tag name.
